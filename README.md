# Orenburg education platform

Сервер имеет несколько режимов запуска: 

```
$ npm run start
```
Запус с ```cluster```, на половину ядер.

```
$ npm run dev
```
Разработка без ```nodemon``` на одно ядро.

```
$ npm run devmon
```
То же самое, но с ```nodemon```.

```
$ npm run sync
```
Синхронизация моделей базы данных.
